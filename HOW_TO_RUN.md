## 1) Enter folder
```bash
cd fitness_agent_site
```

## 2) Create and activate a virtual environment
```bash
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

## 3) Install dependencies
```bash
pip install -r requirements.txt
```

## 4) Run the app
```bash
streamlit run app.py
```

## 5) Deactivate the virtual environment
```bash
deactivate
```
